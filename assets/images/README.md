# ملف توضيحي لصور فريق المشروع

## أسماء الصور المطلوبة والمسار:

```
public/images/
├── team-member-1.jpg    (أحمد محمد - المطور الرئيسي)
├── team-member-2.jpg    (فاطمة علي - مصممة الواجهات)
├── team-member-3.jpg    (محمود حسن - مهندس قواعد البيانات)
└── team-member-4.jpg    (ليلى محمود - مدير المشروع)
```

## الصيغ المدعومة:
- JPG / JPEG (الأفضل للصور الفوتوغرافية)
- PNG (مع خلفية شفافة)
- WEBP (جودة عالية وحجم صغير)

## متطلبات الصور:
- حجم الصورة: 400x400 بكسل على الأقل
- نسبة العرض للارتفاع: 1:1 (مربع)
- الجودة: 85% على الأقل

## ملاحظات:
- الصور ستظهر في إطار دائري بحد أزرق جميل
- إذا لم تجد الصورة، سيتم استخدام صورة افتراضية من UI Avatars
- ضع الصور في مجلد `public/images/`

## كيفية الاستخدام:
1. ضع صور الفريق في المجلد أعلاه
2. الأسماء يجب أن تطابق ما في الكود
3. ستظهر في صفحة About مع دالة fallback للصور الافتراضية
